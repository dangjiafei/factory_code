# 在线互动病例模块产品文档（厂家端）

## 1. 模块概述

### 1.1 功能定位
在线互动病例模块是活动管理下的核心功能模块之一，旨在通过病例互动的方式促进医学知识的传播与学术交流。该模块允许活动发起者创建互动式病例内容，并通过任务分配的方式传递给目标医生群体，实现病例内容的广泛传播与深度互动。

### 1.2 模块入口
**菜单位置**: 活动管理 → 在线病例互动

### 1.3 核心业务场景
- 制药企业创建互动病例
- 通过病例互动形式进行学术推广
- 医生群体参与病例讨论与学习

### 1.4 全局样式调整
- 全局页面内容区的灰色背景颜色统一为” #F8FAFC“
- 一级标签页筛选没有白色背景，就显示标签页名称，选中状态为下划线，下划线颜色为蓝色，下划线小于标签页名称的长度
- 一级标签页没有下划线，只有选中状态的下划线，位置需要舒适，白色背景不能太靠上
- 一级标签页统一字体为 16px，字体颜色” #808080“，选中状态字体颜色” #4A4A4A“
- 白色背景不需要有边框和阴影效果
- 标签页下方为内容区，内容区是白色背景且为圆角，标签页的文字最左边和下方的白色背景左对齐
- 如果有二级标签页，二级标签页是在白色背景里的，并且是以按钮的形式展示，二级标签页整体有一个背景，颜色是”#F8FAFC“，需要有圆角效果
- 二级标签页选中状态为白色背景，有圆角，字体颜色为“ #667EEA”，未选中状态字体颜色为“ #64748B”,二级标签页的文字大小为 14px
- 功能区也是有一个背景，颜色是” #F8FAFC“，需要有圆角效果，里面的功能为白色，字体大小为 14px
- 列表内容的表头背景和上面是一样的，列表中除了操作的按钮文字大小为 12px,其他都是 14px,除非有 ID 显示在标题下方，为 10px
- 列表的表头文字是黑色加粗，标题也是黑色加粗： #2D3748；其他列的字段不加粗，颜色为： #4A5568
- 需要把水印的效果修改的更淡一点，现在有点影响内容区的显示效果

- 以上所有细节均可参考原型图，原型图上体现了所有细节，这个原型图作为我们的标准，以后都按照这个规则执行，如颜色标注有问题，则看原型图上的颜色

## 2. 功能架构

### 2.1 三层子标签结构

| 子标签页 | 功能描述 |
|---------|---------|
| **病例互动学术内容** | 查看当前用户创建的病例互动活动 |
| **病例互动征集活动** | 查看病例互动征集活动 |
| **医生投稿作品库** | 查看医生投稿的病例互动作品 |


#### 2.1.1 病例互动学术内容

- 将列表改为：活动名称、产品标签、互动题目数、创建时间、完成数/总任务数、操作
- 有勾选框，用于批量操作，点击批量分享，如果没有勾选数据，提示用户需要先选择要分享的数据
- 标题下方有 ID 显示在标题下方
- 操作功能改为按钮形式，学术传递使用绿色 #10B98140，查看详情使用浅蓝色， #667EEA40，批量分享使用蓝色，创建在线病例互动使用蓝色，蓝色不是现在系统中的蓝色，而是 #667EEA
- 将搜索框的提示文字改为：搜索活动名称或ID
- 将推广科室和自定义标签收起来，显示更多筛选，点击更多筛选展开/收起更多筛选条件

#### 2.1.2 病例互动征集活动
- 列表字段：活动名称、活动状态、产品标签、活动时间、报名人数、已征集/征集目标数、操作
- 功能区：搜索活动名称或ID，筛选产品标签，筛选活动状态，筛选活动创建范围
- 状态：草稿、审核中、待开始、进行中、已结束、审核失败
- 状态显示不止是文字的显示，需要有背景颜色
- 点击新建内容征集活动，跳转到创建页面

#### 2.1.3 医生投稿作品库
- 列表字段：作品名称、所属活动、产品标签、作者、报名时间、投稿时间、作品状态、完成数/总任务数、操作
- 功能区：搜索作品名称或ID，筛选产品标签，筛选作品状态
- 操作列：
  - 查看详情：调整至作品详情页
  - 更多操作：包含内容合适、内容不合适、学术传递、撤回任务

#### 2.1.4 新建病例互动征集活动
- 活动名称
- 活动详情
  - 有智能模块（按医学领域）
    - 心血管领域、内分泌领域、呼吸系统、神经系统、肿瘤领域、儿科领域
    - 还可输入活动主题关键词，AI 辅助生成活动详情
- 内容要求
  - 系统默认有 3 条，用户可以进行新增或者删除或修改
  - 目标采购数
  - 活动时间
  - 产品标签
  - 活动封面：默认或自定义
- 互动题目设置
  - 提供选项：5、10、自定义
    - 自定义：用户不低于 5 道题目
- 邀约对象筛选条件
  - 和医学播客一样，但是需要将机构名称改为机构等级

#### 2.1.5 病例互动征集活动详情页
- 分块显示
  - 第一块：活动标题、产品标签、活动时间、目标病例互动数、活动状态、病例互动要求
  - 第二块：数据报表：执行报表、代表数据报表、医生数据报表、任务数据报表、答题数据报表
  - 第三块：活动介绍与内容要求
  - 第四块：邀约对象规则
  - 第五块：列表内容：报名客户、作品管理
    - 报名客户列表字段：姓名、身份、职称、科室、机构、省市
    - 作品名称、作者、职称、科室、审核状态、投稿时间、已完成/所有任务、操作（学术传递、撤回任务）
- 返回列表按钮左上角常驻，点击返回列表页
- 右侧按钮：预览活动、编辑、删除、结束
  - 预览活动：弹窗展示
  - 编辑：跳转到编辑页面，修改活动信息
    - 已经开始的活动不能编辑
  - 删除：删除当前活动，确认后从列表中移除
    - 已经开始的活动不能删除
  - 结束：结束当前活动，确认后从列表中移除
    - 已结束的活动不能结束

#### 2.1.6 医生作品详情页
- 分块显示
  - 第一块：封面、标题、状态、产品标签、投稿时间、简介
  - 第二块：数据报表
- 标签页：病例作品、创作者自我评语、分配记录表、互动评论、关联征集活动详情
- 病例作品
  - 展示病例互动的内容，按照步骤展示
- 右上角功能按钮：学术传递、撤回任务
