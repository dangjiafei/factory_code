# 在线互动病例模块产品文档(创作者中心)

## 功能描述

### 新增招募大厅菜单模块
- 顶部为 banner 卡片
  - 2026年度医学学术活动招募
  - 描述：选择符合您专业领域的学术活动，与行业精英共创优质内容，让好内容得到广泛传播，助力医学观念教育
  - 数字：50+ 优质学术活动、3000+ 医生参与、100000+ 内容触达医生

- 底部标签：热门招募活动
- 默认筛选：已关注的活动
- 卡片显示：活动状态：招募中、即将截止（距离开始时间 3 天）、即将开始（活动未开始）
  - 活动标题，最多显示两行
  - 活动类型：医学播客、在线病例互动
  - 活动简介，最多显示四行
  - 截止时间，报名人数
  - 操作按钮：查看详情、立即报名
    - 点击都进入详情页
  
### 活动管理菜单模块
- 医学播客标签页（列表的数据都是报名成功的活动）
  - 列表字段：活动名称、当前状态、活动时间、投稿作品、操作（查看详情）
  - 状态筛选为平铺显示，待投稿后面有数字，表示有未投稿的活动；审核中后面有数字，表示审核中的活动
  - 功能区：搜索活动名称或ID、筛选关注状态
- 在线病例互动标签页（列表的数据都是报名成功的活动）
  - 列表字段：活动名称、当前状态、活动时间、投稿作品】操作（查看详情）
  - 审核状态筛选为平铺显示，待审核后面有数字，表示有未审核的活动
  - 功能区：搜索活动名称或ID，筛选关注状态

### 活动详情页（没有报名的时候）
- 分块显示
  - 第一块：封面、标题、状态、活动时间、收藏图标，立即报名按钮
  - 第二块：活动详情与内容要求
  - 第三块：邀约对象规则

### 活动详情页（已报名的时候）
- 分块显示
  - 第一块：封面、标题、状态、活动时间、收藏图标，去创作按钮
  - 第二块：活动详情与内容要求
  - 第三块：邀约对象规则
  - 第四块：我的作品信息
    - 病例标题、创建时间、当前状态（草稿、待审核、审核成功、审核失败）、操作（详情、编辑、去投稿）
    - 投稿规则说明
  - 第五块：作品学习记录
    - 字段：姓名、职称、科室、机构、学习时间、认证状态
  - 右上角功能：审核通过、审核拒绝

### 创建病例互动内容页面
- 左侧是互动题目要求数量
- 智能辅助功能、文件导入、文本解析
  - 文件导入：用户可以上传文件，系统会自动解析文件中的内容，然后将对应内容填充到右侧的内容区中
  - 文本解析：用户可以将文本粘贴后，点击一键解析，系统会将内容解析到右侧的内容区中

- 右侧是内容区（除病例基本信息外，其他字段都是富文本输入框）
  - 病例基本信息
    - 病例标题（必填）
    - 病例简介（必填）
    - 文献资料（可选，PDF格式文件）
  - 患者基本信息
  - 主诉内容
  - 问诊（现病史、既往史、个人史、家族史）
  - 查体
  - 辅助检查
  - 初步诊断
  - 诊断依据
  - 鉴别诊断
  - 治疗方案
  - 健康指导
  - 自我评语（必填）
- 右上角功能：保存
