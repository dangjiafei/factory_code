# 在线互动病例模块产品文档(CRM)

## 功能描述

- 在工作台模块新增一级菜单“活动管理”
  - 二级菜单：在线病例互动
  - 二级菜单：医学播客

### 在线病例互动菜单模块
- 标签页：互动病例征集活动、医生投稿作品库
- 互动病例征集活动
  - 列表字段：活动名称、活动时间、状态、产品标签、已报名、已采购/目标数、厂家、操作
  - 审核状态筛选为平铺显示，待审核后面有数字，表示有未审核的活动
  - 功能区：搜索活动名称或ID

- 医生投稿作品库
  - 列表字段：病例标题、作者、职称、病例互动采购费、审核提交时间、企业名称、申请组织、审核状态、操作
  - 审核状态：审核状态筛选为平铺显示，待审核后面有数字，表示有未审核的病例
  - 功能区：搜索病例标题、ID或作者，搜索申请组织名称

### 互动病例征集活动详情页
- 如果活动状态为待审核，那么黄色区域提示：“请注意：本次活动处于待审核状态，您需要检查活动内容是否合规合法，是否存在广告嫌疑，是否符合平台规范。审核通过后活动将立即生效。”
- 活动基本信息：
  - 活动名称
  - 产品标签
  - 活动时间
  - 目标病例数
  - 活动状态
  - 病例互动要求
  - 创建企业
  - 创建人
  - 创建时间
- 活动介绍与内容要求
- 邀约对象规则
- 列表内容：标签页：拟推荐客户、作品管理
  - 拟推荐客户：姓名、身份、职称、科室、机构、省市、投稿作品
  - 作品管理：作品名称、作者、职称、科室、机构、审核状态、投稿时间、操作
- 审核记录
  - 字段：审核人、审核时间、审核状态、审核意见
- 右上角功能：审核通过、审核拒绝

### 医生作品详情页
- 活动封面，右边是标题、审核状态、产品标签、投稿时间、简介
- 5 个标签页：病例作品、创作者自我评语、分配记录表、互动评论、关联征集活动详情
